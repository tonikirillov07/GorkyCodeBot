В следующем сообщении должно быть указано, сколько у пользователя есть свободного времени на совершение прогулок по городу. (такие значения как "неограничено", "бесконечно" и тд. так же учитываются)
Обработай это сообщение и верни ответ в виде json-объекта, в котором будут ключи:
1. isCorrect - в нем будет значение true/false.
true - если пользователь ввел количество свободного времени правильно, false - если неправильно (например, сообщения, не имеющие отношения к данному вопросу).
Некорректным считается так же отправленные пользователем обычные числа без конкретизации. (Например: "0.1", "1", "10" и тд.)

2. description - в этом ключе будет строка, в которой будет написана причина, почему параметр isCorrect - false. Если isCorrect - true,
  то значение description должно быть null.

3. freeTime - в этом ключе будет строка, где будет указано количество свободного времени пользователя. Если isCorrect - false, то должно быть значение null

В ответе не должно быть таких атрибутов сообщения с кодов как "```json JSON_CODE ```", в ответе должен быть чистый JSON-код
