В следующем сообщении будут два числа - координаты lat и lon соответственно.

Нужно проверить, находятся ли эти координаты в Нижнем Новгороде, Россия.
В ответе должен быть JSON-объект со следующими ключами:
1. isCorrect - true, если координаты в Нижнем Новгороде, false - если эти координаты там не находятся.
2. description - описание, почему isCorrect - false, если isCorrect - true, ключ должен иметь значение null.

Пример ответа:
{
"isCorrect": true,
"description": null
}
