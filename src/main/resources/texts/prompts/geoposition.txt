В следующем сообщении должно быть указано, где на данный момент находится пользователь в Нижнем Новгороде. (например, адрес, улица или определенные места)
Обработай это сообщение и верни ответ в виде json-объекта, в котором будут ключи:
1. isCorrect - в нем будет значение true/false.
true - если пользователь ввел свое местоположение правильно, false - если неправильно (например, сообщения, не имеющие отношения к данному вопросу или если местоположение, введенное пользователем не находится в Нижнем Новгороде).

2. description - в этом ключе будет строка, в которой будет написана причина, почему параметр isCorrect - false. Если isCorrect - true,
  то значение description должно быть null.

3. userLocation - в этом ключе будет строка, где будет указано местоположение, введенное пользователем. Если isCorrect - false, то должно быть значение null

В ответе не должно быть таких атрибутов сообщения с кодов как "```json JSON_CODE ```", в ответе должен быть чистый JSON-код
