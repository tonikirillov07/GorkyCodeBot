Ниже будут указаны следующие данные:
1. Интересы пользователя
2. Количество свободного времени пользователя
3. Место, где сейчас располагается пользователь (может быть представлена в виде названий улицы, места и тд. или в виде координат lat и lon)

Обработай эти данные и выведи маршрут (3-5 мест), куда может пойти пользователь в соответствии со своими интересами,
удаленностью от этого места, и количеством свободного времени.

Описание одного места должно быть такого вида:
1. Название места
Краткое описание места. (почему именно это место?)

Удаленность: n км./м.
Примерное время: k ч./мин.

В ответе должен быть только список такого вида в JSON-формате, ничего больше.
Можно использовать данные из этой таблицы: %s
Если данных в этой таблице не оказалось, постарайся найти их в интернете.

И важно: ВСЕ НАЙДЕННЫЕ МЕСТА ДОЛЖНЫ НАХОДИТЬСЯ В НИЖНЕМ НОВГОРОДЕ, А НЕ ГДЕ-ЛИБО ЕЩЕ.

Если таких мест по каким-то причинам найти не удалось, то в ответе должна быть указана причина, почему такие места не найдены.

JSON-формат ответа: список из объектов, характеризующих места, а также два параметра.

Два параметра, помимо списка мест:
isCorrect - true, если места найдены, false - если не найдены
description - причина, по которой wasFound - false, если wasFound - true, то этот параметр должен иметь значение null

Ключи для одного json-объекта из списка:
name - название места. Должен иметь значение null, если wasFound - false.
description - краткое описание места. Должен иметь значение null, если wasFound - false.
distance - удаленность места (Это строковый параметр вида: "n км." или "n м."). Должен иметь значение null, если wasFound - false.
duration - примерное время (Это строковый параметр вида: "k ч." или "k мин."). Должен иметь значение null, если wasFound - false.

Сам список должен иметь ключ places

В ответе должен быть ТОЛЬКО JSON-код без каких-либо дополнительных примечаний, текста и тд. Чтобы сообщение было удобным для
парсинга JSON-кода

Навзание места так же должно содержать краткий адрес этого места. Например, "Художественный музей, Кремль, к3 Музей". Информацию можно брать с Google или Yandex карт.

Данные:
Интересы: %s
Свободное время: %s
Местоположение: %s
