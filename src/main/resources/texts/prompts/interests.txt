В следующем сообщении должны быть указаны интересы пользователя в плане мест для посещения в городе (например, стрит-арт, история, кофейни, панорамы и т.д.).
Такие интересы как "красивые места", "безлюдные места", "вайбовые места" и тд. так же должны учитываться.
Обработай это сообщение и верни ответ в виде json-объекта, в котором будут ключи:
1. isCorrect - в нем будет значение true/false.
true - если пользователь ввел свои интересы правильно, false - если неправильно (например, сообщения, не имеющие отношения к данному вопросу).

2. description - в этом ключе будет строка, в которой будет написана причина, почему параметр isCorrect - false. Если isCorrect - true,
то значение description должно быть null.

3. interests - в этом ключе будет краткий список интересов пользователя. Есть isCorrect - false, то должен быть пустой список

В ответе должен быть чистый JSON-код (без дополнительных комментариев), чтобы сообщение было удобно парсить как JSON-код.

Сообщение: %s
